# MonthlySalesGrowth
Data Analytics Project using Python ,SQL and Power BI.

Build an end to end ETL pipeline.I started with raw Excel data, used Python to clean and transform it , stored it centrally in an SQL database,then connected it to Power BI to create a business dashboard for decision-making.
